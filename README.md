# VILLAGE STORIES - Complete YouTube Video Production Checklist

## üìπ PHASE 1: PRE-PRODUCTION & SHOOTING

### Before Shooting:
- [ ] **Plan your content** - Decide the village story/topic (festivals, farming, cooking, daily life, crafts, etc.)
- [ ] **Scout location** - Check lighting, background noise, and interesting backdrops
- [ ] **Charge all equipment** - Phone/camera, backup battery, power bank
- [ ] **Clean camera lens** - Use soft cloth
- [ ] **Check storage space** - Ensure enough space for recording
- [ ] **Plan golden hours** - Shoot during early morning (6-9 AM) or evening (4-7 PM) for best natural light

### During Shooting:
- [ ] **Shoot in landscape mode** (horizontal) - Never vertical for YouTube
- [ ] **Use highest resolution** - 1080p minimum, 4K if possible
- [ ] **Record at 30fps or 60fps** - Consistent frame rate
- [ ] **Hold phone/camera steady** - Use both hands or tripod
- [ ] **Multiple angles** - Get wide shots, medium shots, and close-ups
- [ ] **B-roll footage** - Capture extra clips of village scenery, activities, details (at least 20-30 seconds each)
- [ ] **Record clean audio** - Minimize wind noise, use external mic if possible
- [ ] **Get 5-10 seconds extra** - Start recording before action, end after
- [ ] **Shoot intro/outro separately** - You talking to camera introducing the topic
- [ ] **Capture candid moments** - Natural reactions make great content
- [ ] **Test shots** - Review first few clips for focus, lighting, audio

### Essential Shots for Village Content:
- [ ] Wide establishing shot of village/location
- [ ] Close-ups of hands doing work (cooking, farming, crafts)
- [ ] Faces and expressions of people
- [ ] Tools and traditional equipment
- [ ] Natural elements (trees, animals, fields, water)
- [ ] Cultural details (clothing, decorations, rituals)

## üé¨ PHASE 2: EDITING IN POWERDIRECTOR

### Organization:
- [ ] Transfer all footage to phone/computer
- [ ] Create project folder with date
- [ ] Watch all footage and note best clips
- [ ] Delete blurry or unusable clips

### PowerDirector Editing Steps:

#### 1. Import & Arrange
- [ ] Open PowerDirector
- [ ] Import all clips (tap + button ‚Üí Media)
- [ ] Drag clips to timeline in story order
- [ ] Trim clips - Remove unnecessary parts (long pauses, bad takes)

#### 2. Basic Editing
- [ ] **Cut out dead space** - Keep video moving
- [ ] **Add transitions** - Use simple cuts, occasional dissolve (don't overuse fancy transitions)
- [ ] **Adjust clip speed** - Speed up slow parts (1.5x-2x), slow down action moments (0.5x)
- [ ] **Color correction** - Increase brightness/contrast if needed
- [ ] **Stabilize shaky footage** - Use stabilization tool if available

#### 3. Audio Editing
- [ ] **Remove background noise** - Use noise reduction tool
- [ ] **Balance audio levels** - Keep volume consistent (-3dB to -6dB)
- [ ] **Add background music** - Use copyright-free music (see resources below)
- [ ] **Keep music volume low** - 20-30% so it doesn't overpower dialogue
- [ ] **Add sound effects** - Nature sounds, village ambience if needed
- [ ] **Fade audio in/out** - At beginning and end of music

#### 4. Text & Graphics
- [ ] **Add opening title** - Channel name + video topic (3-5 seconds)
- [ ] **Add subtitles** - For important dialogue or narration
- [ ] **Use text overlays** - Highlight key points or funny moments
- [ ] **Keep text readable** - Large font, white with black outline
- [ ] **Add end screen** - Last 20 seconds with subscribe button

#### 5. Advanced Touches
- [ ] **Add zoom effects** - On important moments
- [ ] **Picture-in-picture** - For reactions or comparisons
- [ ] **Time-lapse** - For long processes (cooking, farming)
- [ ] **Slow motion** - For dramatic moments

#### 6. Export Settings
- [ ] Resolution: **1080p (Full HD)** minimum
- [ ] Frame rate: **30fps or 60fps** (match recording)
- [ ] Bitrate: **High quality** setting
- [ ] Format: **MP4 (H.264)**

**Ideal Video Length:** 8-15 minutes for village content

## üé® PHASE 3: CREATING THUMBNAIL

### Thumbnail Requirements:
- **Size:** 1280 x 720 pixels (16:9 ratio)
- **Format:** JPG or PNG
- **File size:** Under 2MB
- **Safe zone:** Keep important elements in center (avoid edges)

### Thumbnail Design Steps:

#### Using Canva (FREE - Best Option):
1. Go to **Canva.com** or download Canva app
2. Search "YouTube Thumbnail" template
3. Choose village-themed template or blank canvas
4. **Add your video screenshot** - Select best moment from video
5. **Add text overlay:**
   - 3-5 words maximum
   - Large, bold, easy-to-read font
   - Bright colors (Yellow, Red, White work best)
   - Add text shadow or outline
6. **Enhance image:**
   - Increase contrast and saturation
   - Add slight blur to background, keep subject sharp
   - Add arrows or circles to highlight main subject
7. **Include face (if possible)** - Human faces increase clicks
8. Download as PNG

#### Alternative Tools:
- **Pixellab** (Android) - Free, text on images
- **Snapseed** (Android/iOS) - Photo editing
- **Adobe Express** (Web/Mobile) - Free templates

### Thumbnail Best Practices:
- [ ] **Bright and colorful** - Stands out in search results
- [ ] **Clear and uncluttered** - One main focus
- [ ] **Text is readable** - Even on mobile devices
- [ ] **Emotion in faces** - Surprise, joy, curiosity
- [ ] **Consistent branding** - Use similar style/colors for all videos
- [ ] **Avoid clickbait** - Thumbnail should represent actual content
- [ ] **Test on mobile** - Most viewers watch on phones

## üöÄ PHASE 7: UPLOADING TO YOUTUBE

### Upload Checklist:

#### Before Upload:
- [ ] **Video file ready** - Exported in high quality
- [ ] **Thumbnail created** - 1280x720px image ready
- [ ] **Title written** - Engaging and keyword-rich
- [ ] **Description prepared** - Complete with hashtags and links
- [ ] **Tags ready** - 15-20 relevant tags
- [ ] **End screen planned** - Subscribe button + 2 video suggestions

#### During Upload:
1. **Go to YouTube Studio** (studio.youtube.com)
2. **Click "Create" ‚Üí "Upload Video"**
3. **Select your video file**
4. **Add Title** - Paste your prepared title
5. **Add Description** - Paste full description
6. **Upload Thumbnail** - Click "Upload thumbnail" and select your image
7. **Add to Playlist** - Create playlists by topic (e.g., "Village Cooking", "Farming Videos")
8. **Select Audience** - "Not made for kids" (if general audience)
9. **Add Tags** - In "More options" section
10. **Select Category** - "People & Blogs" or "Entertainment"
11. **Add End Screen** - Last 20 seconds of video
12. **Add Cards** - For related videos (optional)

#### Visibility Settings:
- [ ] **Public** - For regular uploads (visible to everyone)
- [ ] **Scheduled** - Set specific date/time for publish
- [ ] **Unlisted** - Only people with link can see (for testing)

#### Advanced Settings:
- [ ] **Allow comments** - Yes (for engagement)
- [ ] **Allow embedding** - Yes
- [ ] **Publish to subscriptions feed** - Yes
- [ ] **Set video location** - Add your village/state
- [ ] **Set video language** - Hindi or English

#### After Upload:
- [ ] **Add to playlists** - Organize your content
- [ ] **Pin a comment** - Asking question to boost engagement
- [ ] **Share on social media** - Instagram, Facebook, WhatsApp
- [ ] **Respond to comments** - First few hours are crucial
- [ ] **Check analytics** - After 24 hours
